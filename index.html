CargoSync 🚛

Real-Time Freight Intelligence Platform

The Canadian freight industry runs on phone calls, spreadsheets, and outdated TMS software built in the 2000s. CargoSync changes that. It's a production-grade, multi-tenant SaaS platform built for freight brokers — featuring live GPS tracking with geofencing, AI-powered Bill of Lading processing that extracts structured data from photos, a predictive ETA engine that flags at-risk deliveries before they're late, and an event-driven architecture using Apache Kafka that powers every downstream action automatically. Each brokerage gets isolated data, custom roles, and their own dashboard. This isn't a side project — it's a real system solving a real market gap.

E-Commerce Platform 🛒

Full-Stack MERN Shopping Application

A complete full-stack e-commerce application built with the MERN stack — not a tutorial clone, but a production-ready system with real architecture decisions. Features role-based authentication with JWT, Stripe payment gateway with checkout sessions and webhooks for real-time order fulfillment, a smart cart system with localStorage and database sync ensuring zero cart loss during guest-to-user transitions, and server-side search, filtering, sorting, and pagination for scalable product browsing. Data fetching is optimized with React Query for caching and background sync — keeping the UI fast and responsive at scale.

Career Portal 💼

Role-Based Job Platform

A role-based job platform that handles three distinct user types — job seekers, employers, and admins — each with their own dashboard, permissions, and workflows. Built with JWT authentication and refresh token flow for secure, seamless sessions. Employers can post jobs, track applicants, and manage hiring pipelines in real time. Seekers can upload resumes and profile photos via Cloudinary and Multer for scalable cloud-based file management. Every API endpoint is protected with rate limiting, owner-based access control, and protected routes — making it production-secure from day one.
